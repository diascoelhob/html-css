Itálico / Enfase

Nesta frase, temos um termo em Itálico usando a tag i (Não semántica)

Nesta frase, temos um termo em enfase usando a tag em (semantico)

Recomendação

Não use tags não semanticas (i, b)

Use tags semanticas (strong = destaque; em = enfase)
